Creative coding of visuals elements and games

Category

Academic Project

Role

UX/UI

Project Context

Throughout a 5-day bootcamp, I learned how to to code games or visual elements using Processing, a software sketchbook and programming language based on Java that aims to help people code in the context of the visual arts.

First Sketches

The aim of the bootcamp was to code and present a visual element, known as a sketch within Processing, at the end of each day. It was a steep learning curve and my first time coding. The very first sketch focused on creating a dynamic drawing, a concept that changes everytime the program is run, yet it is the same. Click on the arrow in the center of the sketch to run the code.

The second day I coded a game inspired on the classic "Snake". Truth be told, it was a failure as I didn't manage then dynamize it, but it was the opportunity to learn from it! Below you can play it. Just press the key "d" to move to the right, "a" to move left, "w" to move upwards and "s" to downards.

The third sketch focused on creating an interactive pattern, a visual element that would be dynamic and required the user's input. Hover your mouse over the sketch and discover how it reacts to its movement.

Featured Game

The game I coded was inspired from the classic "Frog". Below you should be able to play it. Press the key "d" to move forward and "a" to move backwards.
